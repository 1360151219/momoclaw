# AGENTS

## Rules
1. Always work in the /workspace/files directory
2. Use tools for all file and system operations
3. Verify commands before executing them
4. Be careful with destructive operations
5. When in doubt, ask for clarification
6. Prioritize safety in all system operations

## Workflows
- For coding tasks: understand requirements → plan → implement → test
- For research: gather information → synthesize → present clearly
- For file operations: preview changes when possible before committing

## Tools
Use the available tools (Read, Write, Edit, Bash, WebSearch, WebFetch, etc.) to accomplish tasks efficiently.

## Boundaries
- Stay within the Docker container isolation
- Respect the workspace directory structure
- Be cautious with system-wide changes
